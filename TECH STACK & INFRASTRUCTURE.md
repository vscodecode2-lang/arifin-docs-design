```
# TECH STACK & INFRASTRUCTURE
```

```
## Frontend
```

```
- Framework: Next.js (App Router)
```

- `Bahasa: TypeScript` 

- `Styling: Tailwind CSS` 

- `Komponen UI: Radix UI / Shadcn UI (opsional untuk form dan elemen interaktif yang cepat)` 

- `Form Handling: React Hook Form & Zod (untuk validasi data yang ketat)` 

```
## Backend & Database (BaaS)
```

- `Platform: Supabase` 

- `Database: PostgreSQL` 

- `Authentication: Supabase Auth (Email & Password untuk Admin Dashboard)` 

- `Security: Row Level Security (RLS) diaktifkan untuk semua tabel.` 

```
## Deployment
```

- `Frontend Hosting: Vercel` 

- `Version Control: GitHub`
